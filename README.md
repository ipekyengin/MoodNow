# MoodNow

MoodNow is an AI-powered mobile application that recommends personalized content based on the user's current mood. By leveraging intelligent prompt generation and external APIs, the app delivers tailored movie suggestions along with detailed information and user interaction features.

[Youtube Link](https://youtu.be/Add_X-ZSV2A)

---

## Features

- **AI-Based Mood Analysis**  
  Generates personalized content recommendations using AI-driven prompts.

- **Movie Recommendations**  
  Suggests movies based on the user's mood and preferences.

- **TMDB API Integration**  
  Fetches up-to-date movie data including descriptions, posters, and metadata.

- **IMDb Ratings**  
  Displays IMDb scores to help users evaluate recommendations.

- **My Shows List**  
  Allows users to save and manage their favorite movies.

- **User Reviews**  
  Users can write and view reviews for movies.

- **Filtering System**  
  Advanced filtering options for better content discovery.

---

## Technologies Used

- **Flutter** – Cross-platform mobile development  
- **Dart** – Application logic  
- **REST API Integration** – TMDB API  
- **AI Prompt Engineering** – Mood-based recommendation logic

## ⚙️ Installation

```bash
# Clone the repository
git clone https://github.com/your-username/moodnow.git

# Navigate to project folder
cd moodnow

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## API Configuration

To run this project, you need to obtain an API key from TMDB:

1. Go to https://www.themoviedb.org/
2. Create an account and generate your API key
3. Add your API key to the project:

```dart
const String apiKey = "YOUR_API_KEY";
```
